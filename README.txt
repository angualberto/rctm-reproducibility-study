Resultados dos testes de aleatoriedade e estrutura para rctm_all.bin

- compressao.txt: Teste de compressão (gzip)
- ent.txt: Teste ENT (entropia, chi-square, correlação)
- full_report.txt: Teste Dieharder completo
- smallcrush.txt: TestU01 SmallCrush (se disponível)
- periodo.txt: Teste de período (repetição de blocos)
- linear_complexity.txt: Complexidade linear (Berlekamp–Massey)
- correlacao_alta_ordem.txt: Correlação de ordem alta

Cada arquivo contém o resultado bruto do respectivo teste.

Para análise, abra cada arquivo individualmente.
